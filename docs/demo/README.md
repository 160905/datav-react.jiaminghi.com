# DEMO

::: tip TIP
Demo 均使用了[全屏容器](/guide/fullScreenContainer.html)进行内容缩放，查看时请 F11 全屏以便于 Demo 充满屏幕。

Mac 的 Retina 及其他高分辨率屏幕下查看 Demo 可能出现缩放异常，如出现异常请调节屏幕至最大分辨率。
:::
